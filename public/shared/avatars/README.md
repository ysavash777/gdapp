# Avatares

Coloca aquí las imágenes en formato **.jpg**, con estos nombres exactos:

- `avatar.jpg` — predeterminado (se usa si el usuario no eligió ninguno)
- `avatar-1.jpg`
- `avatar-2.jpg`
- `avatar-3.jpg`
- `avatar-4.jpg`
- `avatar-5.jpg`

Mientras un archivo no exista, se muestra automáticamente la inicial del usuario en su lugar.
